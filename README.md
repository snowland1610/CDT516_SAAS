# 高校教务协作 SAAS（演示）

三端（教务处、教师、学生）教务协作演示项目：小组组队、课表日历、任务/公告等。技术栈：Next.js 14（App Router）+ Tailwind CSS + Supabase。

## 快速开始

1. 安装依赖：`npm install`
2. 配置 `.env.local`：`NEXT_PUBLIC_SUPABASE_URL`、`NEXT_PUBLIC_SUPABASE_ANON_KEY`
3. 启动：`npm run dev`
