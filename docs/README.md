# 项目文档索引

本目录集中存放项目说明、规划与需求文档，便于查阅与交接。

---

## 目录结构

| 目录 | 用途 | 主要文档 |
|------|------|----------|
| **handover/** | 对话交接、当前进度、未完成事宜 | 对话交接-当前进度与未完成事宜.md、对话交接-项目进度与后续步骤.md |
| **planning/** | 开发顺序、阶段规划、页面与需求 | 程序结构与页面生成顺序规划.md、后续工作规划与顺序.md、页面设计需求文档.md、阶段6-公告功能开发计划.md |
| **product/** | 产品说明、功能列表、演示流程 | 三端功能列表与演示流程.md |
| **features/calendar/** | 课表相关需求与实现计划 | 课表周/日/月视图需求与计划文档 |
| **reference/** | 数据结构、环境与同步说明 | 数据结构与数据库设计-演示Mock.md、附录A-Supabase环境配置.md、本地数据库与云端同步说明.md |
| **design/** | UI 风格与改动计划 | UI美化改动计划.md（Notion+Linear 风格、分阶段实施） |

---

## 新对话 / 新人建议

1. 先读 **handover/** 下的交接文档，了解当前进度与待办。
2. 做功能前查阅 **planning/** 中的阶段划分与页面设计。
3. 课表、公告、任务等具体需求在 **features/** 或 **planning/** 中对应文档。
4. 做 UI 美化时以 **design/UI美化改动计划.md** 为唯一依据，按阶段执行。

代码与库表路径见交接文档中的「关键文件与结构」表；所有路径均以项目根目录为基准。
