# 课表月视图 - 交付效果与实现计划

> 在现有周/日视图基础上增加「月视图」：月网格展示当月及前后补全的日期、每日有课数量或小色块，点击某日切到日视图并定位到该日。供后续编程实现时对照。

---

## 一、交付效果（可验收）

### 1.1 视图模式扩展

- **入口**：仍为学生端 `/student/calendar`、教师端 `/teacher/calendar`，不新增路由。
- **视图模式**：由「周」「日」扩展为「周」「日」「月」三选一（如 Tab 或按钮组）。
- **选「月」**：展示月视图（月网格 + 每日有课信息）；选「周」「日」时行为与现有一致。

### 1.2 月视图布局

- **月网格**：按「周」为行、「星期几」为列（周一～周日，与周视图一致），展示一个月的日期。
  - 第一行可为上月末尾几天（补全首周），中间为当月 1 日～月末，最后一行可为下月开头几天（补全末周），使网格规整为若干整周（通常 5～6 行 × 7 列）。
- **每个格子**：表示某一天，需包含：
  - **日期数字**：如 1、2、…、28；非当月日期可用灰色或弱化样式区分。
  - **有课信息**：当日有排课时，展示「有课」的提示，例如：
    - 方案 A：显示数量，如「3 节」；
    - 方案 B：若干小色块/圆点（按课程区分颜色，与周/日视图配色一致）；
    - 方案 C：数量 + 最多 1～2 个课程名缩写或首字（可选，视格子空间而定）。
  - 推荐先做「数量」或「小色块」，保证可读且与 `getCourseColorClasses(courseId)` 一致。
- **今天**：当前日期在网格中可高亮（如边框或背景），便于识别。

### 1.3 月内每日事件计算

- **数据来源**：复用现有 `scheduleRows`，不新增接口。
- **规则**：对月网格中出现的**每一个日期**（含上月尾、下月头）：
  - 得到该日期的 `dateStr`（YYYY-MM-DD）和星期几 `dayOfWeek`（1～7）；
  - 从 `scheduleRows` 中筛出：`row.day_of_week === dayOfWeek` 且 `dateStr >= row.valid_from && dateStr <= row.valid_to` 且 `row.section?.course` 存在；
  - 统计条数即「该日有课数」；若需小色块，则按 `course_id` 去重得到课程列表，每个课程对应一种颜色（复用 `getCourseColorClasses`）。
- **与日视图一致**：同一日在月视图显示的「有课」与切换到日视图后该日的事件列表一致（同一套 `valid_from`/`valid_to` + `day_of_week` 逻辑）。

### 1.4 与日视图的联动

- **点击某日**：用户点击月视图中某一日期格子时：
  - 将「当前选中日期」`selectedDate` 设为该格子对应的日期；
  - 将视图模式 `viewMode` 切换为 `'day'`；
  - 页面即展示日视图，且展示的为刚点击的那一天（与「上一日」「下一日」行为一致）。
- **可选**：若当前已在日视图且 `selectedDate` 为某日，从日视图切回月视图时，可将「当前月」定位到 `selectedDate` 所在月，便于对照。

### 1.5 月视图导航

- **当前月**：月视图下有一个「当前展示月份」状态（如 `monthStart`：当月 1 日 0 点，或 `currentYear` + `currentMonth`）。
- **操作**：
  - 「上一月」「下一月」：切换展示的月份；
  - 可选：「本月」按钮，回到当前自然月；
  - 顶部展示月份标题，如「2026年2月」。

### 1.6 空态与无课日

- **无课日**：当日无排课时，格子内仅显示日期数字，不显示「0节」或色块。
- **空月**：若整月无课，月网格仍正常显示日期与导航，仅所有格子无有课提示即可。

## 二、实现计划（供编程依据）

### 2.1 状态设计

| 状态 | 类型 | 说明 |
|------|------|------|
| `viewMode` | `'week' \| 'day' \| 'month'` | 当前视图模式，新增 `'month'` |
| `monthStart` | `Date` | 月视图下「当前展示月份的 1 日 0 点」；用于生成月网格及上一月/下一月 |
| `weekStart` | `Date` | 周视图用（已有） |
| `selectedDate` | `Date` | 日视图用；从月视图点击某日时写入该日并切到日视图（已有） |

### 2.2 月网格日期生成

- 由 `monthStart`（当月 1 日）得到：
  - 当月 1 日的星期几 `getDayOfWeek(monthStart)`；
  - 当月总天数（根据 year/month 计算）；
- 生成「月网格」的日期数组（约 5～6 周 × 7 天）：
  - 第一格：从「当月 1 日所在周的周一」开始（若 1 日为周三，则第一格为上周一）；
  - 按顺序填满 7×6 或 7×5 格，每格一个 `Date`；
  - 或：先算当月 1 日所在周一的日期，再生成 42 个连续日期（6 周），便于循环渲染。
- 每格用 `formatDate(cellDate)` 得到 `dateStr`，用于与 `valid_from`/`valid_to` 比较及传给日视图。

### 2.3 每日事件计算（月内）

- **输入**：`scheduleRows`、当月网格中每个格子的 `Date`（或 `dateStr`）。
- **对每个格子**：
  - `dayOfWeek = getDayOfWeek(cellDate)`，`dateStr = formatDate(cellDate)`；
  - 过滤 `scheduleRows`：`row.day_of_week === dayOfWeek` 且 `dateStr >= row.valid_from` 且 `dateStr <= row.valid_to` 且 `row.section?.course` 存在；
  - 得到该日事件列表（可与日视图共用同一「转成事件对象」的逻辑，仅不排序也可）；
  - 数量 = 列表长度；若做小色块，按 `course_id` 去重得到课程 ID 列表，每个用 `getCourseColorClasses(courseId)` 取色。
- **结构**：可用 `useMemo` 计算 `monthGridEvents: Record<string, { count: number; courseIds: number[] }>`，key 为 `dateStr`，便于按格子渲染。

### 2.4 月视图 UI 结构

- **顶部**：视图切换「周」「日」「月」；当 `viewMode === 'month'` 时显示「上一月」「下一月」「本月」及标题「YYYY年M月」。
- **内容**：
  - 表头一行：周一～周日（与周视图一致的 `DAY_NAMES` 去掉空串或取 1～7）；
  - 下方若干行：每行 7 个格子，每格一个 `Date`；格内显示日期数字 + 有课时的数量或小色块；今日高亮；点击格子 → `setSelectedDate(该格 Date)` 且 `setViewMode('day')`。
- **样式**：格子可用 `border`、`min-height` 统一；非当月日期 `date.getMonth() !== monthStart.getMonth()` 时文字灰色；当日加 `ring` 或背景色。

### 2.5 与日视图联动细节

- 点击月格：`setSelectedDate(new Date(cellYear, cellMonth, cellDate))`（注意用本地日期，避免时区问题），然后 `setViewMode('day')`。
- 日 → 月：若希望从日视图切回月视图时「当前月」为选中日所在月，可在切换为月视图时执行 `setMonthStart(selectedDate 所在月 1 日)`（或在点击「月」时根据 `selectedDate` 初始化 `monthStart`）。

### 2.6 可复用与不涉及

| 项 | 说明 |
|----|------|
| `scheduleRows`、fetch | 完全复用 |
| `formatDate`、`getDayOfWeek`、`getCourseColorClasses` | 复用 |
| `valid_from` / `valid_to` 过滤逻辑 | 与周/日视图一致 |
| 事件详情弹窗、悬停 | 月视图格子可不做详情弹窗，仅点击切日视图；若需「点击某格弹当日课程列表」可后续加 |
| 不涉及 | 新接口、新表、新路由 |

### 2.7 实现顺序建议

1. **状态与切换**：`viewMode` 增加 `'month'`；增加 `monthStart` 状态；月视图下展示「上一月」「下一月」「本月」和月份标题。
2. **月网格生成**：由 `monthStart` 生成 6 周 × 7 的日期二维数组（或一维 42 个 Date），表头周一～周日，渲染空格子并显示日期数字，非当月灰色、今日高亮。
3. **每日事件计算**：`useMemo` 内对网格中每个 `dateStr` 统计有课数及课程 ID 列表，得到 `monthGridEvents`。
4. **格子内有课展示**：在每格内根据 `monthGridEvents[dateStr]` 显示数量或小色块。
5. **点击格子 → 日视图**：格子点击时设置 `selectedDate` 并 `setViewMode('day')`。
6. **可选**：从日视图切到月视图时，用 `selectedDate` 初始化或同步 `monthStart`。

### 2.8 文件与范围

- **改动**：仅 `app/student/calendar/page.tsx`、`app/teacher/calendar/page.tsx`。
- **不改**：`lib/calendarGrid.ts`、`lib/calendarColors.ts`（仅复用），不新增组件文件也可（逻辑简单则内联在页内）。
