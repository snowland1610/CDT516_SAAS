### 课表日历（学生 + 教师）——已全部完成

- **入口**：学生/教师侧栏「课表」→ `/student/calendar`、`/teacher/calendar`。
- **数据**：学生 = 本人选课 section 的 section_schedule；教师 = 本人负责课程的 section_schedule。过滤规则：`day_of_week` + 日期在 `valid_from`～`valid_to` 内。
- **周视图**：
  - 时间网格 08:00～18:00、30 分钟一格；横线 + 竖线网格；事件块按 `eventToGridRow`、`getCourseColorClasses(courseId)` 着色。
  - 颜色：Tailwind `content` 已包含 `./lib/**`，课程色不被 purge；`calendarColors.ts` 中 courseId 为空时降级灰色。
  - 色块时长：`lib/calendarGrid.ts` 按持续分钟数算 span，结束整点按「到下一半点」处理；2.5 小时且从半点开始时按 3 小时算，避免 8:30～11:30 只画到 11:00。
- **日视图**：
  - 视图切换「周 / 日 / 月」；日视图下 `selectedDate`，导航「上一日」「下一日」「今天」，单列时间轴 + 当日事件；空态「当日暂无课程」。
  - 数据：`dayEvents` 由 `scheduleRows` + `selectedDate`（星期几 + 日期在 valid 内）过滤，与周视图同逻辑。
- **月视图**：
  - `monthStart`、6 周×7 日月网格；`monthGridEvents` 为每个日期算 count 与 courseIds，格子内小色点 +「n 节」。
  - **交互**：无课日不响应点击、仅悬停；悬停显示当日事件预览（课程名 + 时间）；有课日点击打开「当日详情」弹窗，弹窗内简单列表（课程名、时间）+「进入日视图」按钮，点击后切到日视图并定位该日。
  - 辅助：`getDayEventsForDate(rows, date)` 供月视图悬停与弹窗复用；状态 `hoverCellDate`/`hoverCellPos`、`selectedCellDate`。
- **工具**：`lib/calendarGrid.ts`（`eventToGridRow`、`timeToSlotIndex`、`getSlotLabel`）、`lib/calendarColors.ts`（`getCourseColorClasses`）。
